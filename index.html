Cloud Engineering Bootcamp 2026

Design. Deploy. Scale.

About the Event

The Cloud Engineering Bootcamp 2026 is a hands-on event designed to introduce students to real-world cloud computing concepts and tools. Participants will learn how to design, build, and deploy scalable applications using cloud services such as AWS. The event is intended for Computer Science and IT students, especially those interested in becoming cloud engineers or learning more about modern infrastructure. No prior cloud experience is required, making it accessible to beginners while still valuable for those with some technical background.

What To Expect

Hands-On Learning, Real-World Cloud Experience

  • Build Real Projects: Deploy applications using AWS, Docker, and cloud tools.
  • Collaborative Learning: Work with peers on team-based projects and problem-solving.
  • Guided Support: Get help from instructors and mentors throughout the program.
  • Career Preparation: Gain skills needed for entry-level cloud engineering roles.

Event Schedule

Start Date: June 1, 2026

End Date: August 21, 2026

Class Times: Tuesdays & Thursdays, 6:00 PM – 8:00 PM

Weekend Labs: Saturdays, 11:00 AM – 1:00 PM

Week 1–2: Cloud Basics

Introduction to cloud computing, networking basics, service models (IaaS, PaaS, SaaS), and understanding how cloud systems work.

Week 3–5: AWS + Azure

Hands-on learning with AWS and Microsoft Azure, including storage, virtual machines, IAM, and cloud deployment tools.

Week 6–8: DevOps

Learn GitHub, Docker, CI/CD pipelines, automation, and team collaboration workflows.

Week 9–12: Final Project

Build and deploy a real cloud-based application, then present your project portfolio.
