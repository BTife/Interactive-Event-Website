# Interactive-Event-Website

## Overview

This project is a responsive web application built using HTML, CSS, and JavaScript. It demonstrates core front-end development skills including layout design, DOM manipulation, and user interaction.

## Features

* Dark mode toggle
* RSVP form with validation
* Dynamic participant counter
* Responsive design using Flexbox
* Embedded media content

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
index.html
styles.css
script.js
assets/
```

## What I Learned

* How to structure a responsive website
* DOM manipulation using JavaScript
* Form validation and event handling
* Styling with CSS Flexbox
